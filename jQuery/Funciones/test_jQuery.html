AddClass

Agregar la clase a uno o mas elementos.

Append

Inserte contenido, especificado por el parámetro, al final de cada elemento en el conjunto de elementos coincidentes.

Click

Vincula un controlador de eventos al evento de JavaScript "clic" o activa ese evento en un elemento.(Ejemplo, activar una alerta)

Hide

Ocultar los elementos coincidentes.
Como por ejemplo este párrafo

Show

Muestra los elementos coincidentes.
Como por ejemplo el párrafo ocultado arriba ;)

Toggle

Muestre u oculte los elementos coincidentes.
Como este h1 y p.

SlideUp

Oculte los elementos emparejados con un movimiento deslizante.

SlideDown

Muestre los elementos emparejados con un movimiento deslizante.

SlideToggle

Muestre u oculte los elementos emparejados con un movimiento deslizante.
Como todos los parrafos de arriba.

Fadeout

El efecto fadeOut() oculta un elemento variando su opacidad, veamos su sintaxis extraída de su web oficial.

Fadein

muestra un elemento (anteriormente oculto) cambiando su opacidad, aquí tenéis su sintaxis extraída de su web oficial.

After

Su funcionalidad es la de añadir elementos después de otro elemento indicado dentro del DOM de una página web.

Before

Su funcionalidad es la de añadir elementos por delante de otros ya existentes dentro del DOM de una página web.

Html

podemos obtener código html de un nodo del DOM o asignárselo.
De esta manera podremos manipular de una manera fácil los diferentes elementos que hay en una página web.

Attr

vale para obtener y/o modificar valores en los atributos de los diferentes elementos del DOM de una página web.

Val

podemos asignar u obtener valores de ciertos elementos del DOM de una web, estos elementos son los inputs de formularios y las etiquetas select y textarea.

Text

Con esta manera podemos obtener o asignar el texto que queramos a los diferentes elementos del DOM.
